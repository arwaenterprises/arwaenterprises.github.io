The three roles

Right Access for Every User

👑 Admin
All departments — full view
Approve / reject LOPs
Freeze payroll periods
Generate billing reports
Manage employees & roles
👤 Supervisor
Own department only
Raise LOP (needs admin approval)
Submit OT hours
View attendance records
Cannot freeze or export
📟 Punch Terminal
Public URL — no password
Face punch only
No reports, no settings
Works offline (PWA)
Ideal for shared devices
